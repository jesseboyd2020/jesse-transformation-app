GITHUB PAGES UPDATE
1. Extract this ZIP.
2. Open your GitHub repository.
3. Replace the old index.html, manifest.json, sw.js and icon files with these files.
4. Commit the changes to main.
5. Wait for GitHub Pages to deploy.
6. On iPhone, close and reopen the app. If the old version remains, remove it from the Home Screen, open the website in Safari, refresh, and add it to the Home Screen again.

DATA
Your saved data stays in Safari local storage as long as the GitHub Pages address and storage key remain unchanged. Export a backup before updating.
