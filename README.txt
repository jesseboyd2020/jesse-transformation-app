JESSE TRANSFORMATION ELITE v3

WHAT IS INCLUDED
- Daily dashboard
- Day-by-day workout plan
- Breakfast, lunch, pre-workout, post-workout, dinner and before-bed macro goals
- Workout set logging and rest timer
- Meal completion and meal notes
- 84-day calendar
- Water, habits, weight, sleep and steps
- Measurement charts
- Progress photos
- Automated coaching feedback
- Data backup and restore
- iPhone Home Screen support and offline caching

IPHONE INSTALL
1. Upload every extracted file to an HTTPS host such as Netlify, GitHub Pages, Vercel or your own website.
2. Open the hosted link in Safari.
3. Tap Share > Add to Home Screen.

WINDOWS / DESKTOP
Open index.html directly. Hosting it is preferred because offline caching and Home Screen installation require HTTPS or localhost.

DATA
All records are stored locally in the browser. Export backups regularly. Clearing browser website data can erase records.

CURRENT LIMITATIONS
This build does not contain live barcode scanning, Apple Health/Watch syncing, cloud accounts, push notifications, commercial AI APIs or App Store signing. Those require backend services, external APIs and Apple developer credentials.


V3 UPDATES

- Unchecking a meal instantly removes its macros.
- Macro progress bars show totals and remaining targets.
- Meal completion is saved immediately.
- Improved premium visual design, spacing, cards and mobile presentation.
